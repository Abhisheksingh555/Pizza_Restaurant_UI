## 📌 Project Features

- ✅ Responsive layout with semantic HTML5 structure
- ✅ Navigation with ARIA accessibility
- ✅ Font Awesome social media icons
- ✅ SEO-optimized `<meta>` tags
- ✅ Multiple themed sections:
  - Hero Section
  - About Us
  - Featured Menu
  - CEO’s Message
  - Why Choose Us
  - Contact Info
  - Footer with Quick Links & Social Icons
